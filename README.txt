############********** Pulse Data Table Project **********###############
This project contains the follow:
	- a folder containing code for the client
	- a folder containing code for the server
	- a bash shell file to execute the application. 

INSTRUCTIONS TO RUN::

 - Make sure you do not have anything currently running on ports Localhost:3000 or 
   Localhost:3001
 - Open 2 terminals.
 - cd into the directory this project is in on both of them.
 - In the first terminal type the command "bash server.sh"
 - In the second terminal type the command "bash client.sh"
 - Open your browser to localhost:3000
